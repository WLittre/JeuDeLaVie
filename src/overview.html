Jeu de la Vie

Implementation du Jeu de la Vie de Conway en Java utilisant plusieurs Design Patterns :

  • Pattern Etat : Gestion des etats vivant/mort d'une cellule via l'interface CelluleEtat et ses implementations CelluleEtatVivant / CelluleEtatMort, avec un Singleton pour les instances uniques.
  • Pattern Observateur : JeuDeLaVie implemente Observable et notifie ses observateurs (JeuDeLaVieUI, ObservateurConsole) a chaque changement de generation.
  • Pattern Commande : Les transitions de cellules sont encapsulees dans des commandes (CommandeVit, CommandeMeurt) executees en une seule passe apres le calcul.
  • Pattern Visiteur : Les regles du jeu sont implementees dans des visiteurs interchangeables (VisiteurClassique, VisiteurHighLife, VisiteurDayNight, VisiteurVieSansMort, VisiteurCovid).

Le projet inclut egalement un systeme de sauvegarde/chargement de structures via les classes Structure et Gestionstructure.
